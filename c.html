C Programming Language

C is a procedural programming language. It was initially developed by Dennis Ritchie as a system programming language to write operating system. The main features of C language include low-level access to memory, simple set of keywords, and clean style, these features make C language suitable for system programming like operating system or compiler development.

Basic of C

C is a general-purpose high level language that was originally developed by Dennis Ritchie for the Unix operating system.

Funtion

A function is a group of statements that together perform a task. Every C program has at least one function

Variable

A variable is a name of the memory location. It is used to store data. Its value can be changed, and it can be reused many times.

Pointers

The pointer in C language is a variable which stores the address of another variable. This variable can be of type int, char and etc.

Data Types

A data-type in C programming is a set of values and is determined to act on those values. C provides various types of data-types.

Storage Classes

We use the storage class in the C language for determining the visibility, lifetime, initial value, and memory location of any given variable.

Enum, Struct and Union

C provides special types called enumerated types or enum to declare symbolic names that represent integer constants.

Input / Output

Input and output functions are available in the C language to perform the most common tasks. In every C program, three basic functions take place – accepting of data as input.The acceptance of data refers to input.

Memory Management

C language provides many functions that come in header files to deal with allocating and managing memories.
