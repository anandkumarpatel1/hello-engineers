Java

Java IDL is a technology for distributed objects – that is, objects interacting on different platforms across a network. Java IDL enables objects to interact regardless of whether they're written in the Java programming language or another language such as C, C++, COBOL, or others.

Overview of Java

JAVA was developed by James Gosling at Sun Microsystems Inc in the year 1995, later acquired by Oracle Corporation. It is a simple programming language.

Packages in Java

A java package is a group of similar types of classes, interfaces and sub-packages. Package in java can be categorized in two form, built-in package and user-defined package.

Basics of Java

Java program is a collection of objects, and these objects communicate through method calls to each other to work together. Classes and Objects, Method, Instance variables, syntax, and semantics of Java.

Collection Framework

Any group of individual objects which are represented as a single unit is known as the collection of the objects. In Java, a separate framework named the “Collection Framework”.

Flow Control in Java

Java is a programming language which is uses control statements to control the flow of execution of a program based on certain conditions. These are used to cause the flow of execution to advance and branch based on changes to the state of a program.

List

ArrayList is a part of the Java collection framework and it is a class of java.util package. It provides us with dynamic arrays in Java.

Queue

The AbstractQueue class in Java is a part of the Java Collection Framework and implements the Collection interface and the AbstractCollection class.

Map
